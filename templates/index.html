Cats&Dogs

API
for cats:
1. /api/cats/username/ : List all cats (Also accept POST and PUT)
2. /api/dogs/username/1 : Returns cat if its id is 1 (Also accept DELETE)
for dogs:
1. /api/dogs/username/ : List all cats (Also accept POST and PUT)
2. /api/dogs/username/1 : Returns cat if its id is 1 (Also accept DELETE)
